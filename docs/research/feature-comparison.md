# Selected MVP Features

The following features will be implemented:

1. Salary Submission System
2. Validation & Confidence Scoring
3. Company Name Normalization
4. Duplicate Detection
5. Salary Search & Filtering
6. Compensation Comparison Engine
7. Company Compensation Analytics
8. Market Benchmarking
9. Underpaid/Overpaid Analysis

---

# Deferred Features

The following features are intentionally excluded from the MVP:

* Job Listings
* Company Reviews
* Interview Experiences
* Social Discussions
* Salary Trend Forecasting

These features do not directly contribute to the core objective of building a Compensation Intelligence System and would increase implementation complexity without improving backend evaluation outcomes.
